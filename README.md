
## 🚀 Deployment

This project is hosted via **GitHub Pages**:  
👉 [Live Site](https://allan-77.github.io/myshop-ecommerce/)  

To deploy:
1. Clone the repo
2. Push updates to `main` branch
3. GitHub Pages auto-publishes from `/root`
